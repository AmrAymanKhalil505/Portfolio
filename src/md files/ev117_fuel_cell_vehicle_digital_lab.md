# EV117 Fuel Cell Vehicle Digital Lab

## Category
Educational Simulation / Energy Systems

## Short Summary
Unity-based educational simulation for a fuel-cell vehicle trainer, visualizing hydrogen fuel-cell operation, hybrid energy flow, battery support, monitoring data, and vehicle testing behavior.

## Portfolio Positioning
Present this as an energy/automotive educational simulation.

This project should show that I can build simulations beyond normal games:
- renewable energy education
- hydrogen fuel-cell systems
- automotive training
- interactive monitoring dashboards
- system behavior visualization

## Main Learning Goal
The user should understand how a fuel-cell vehicle system works at a high level:
- hydrogen is supplied to the fuel cell
- the fuel cell generates electrical power
- the battery supports the system
- the motor/vehicle consumes power
- monitoring data helps analyze performance

## What I Built
- 3D virtual fuel-cell vehicle trainer
- Fuel cell system visualization
- Hydrogen supply / regulator behavior visualization
- Battery and power-flow display
- Monitoring dashboard
- Vehicle/test-bench behavior
- Guided educational workflow
- Media/demo-ready presentation for portfolio

## Simulated Components
- Fuel cell stack
- Hydrogen storage / hydro-fill station
- Pressure switch
- Pressure regulator
- Monitoring board
- Battery
- Testing bench
- Remote control
- Power supply
- Vehicle body/chassis
- Motor / drivetrain visualization
- Energy-flow dashboard

## Simulated Behaviors
- Hydrogen supply into the fuel-cell system
- Pressure regulation before fuel-cell input
- Fuel cell generating electrical power
- Battery support / hybrid energy behavior
- Vehicle motor consuming power
- Test bench / running behavior
- Monitoring board values changing during operation
- Energy-flow visualization between fuel cell, battery, and motor

## Feature Cards

### Fuel Cell Stack Visualization
Shows the fuel-cell stack as the main energy-generation unit.

### Hydrogen Supply Path
Visualizes hydrogen storage/filling, pressure switching, and pressure regulation.

### Hybrid Energy Flow
Shows how fuel cell power and battery power support vehicle operation.

### Monitoring Dashboard
Displays simulated system data such as power, voltage, current, pressure, and operating state.

### Test Bench Mode
Shows the vehicle running or being tested in a controlled trainer setup.

### Guided Learning Mode
Explains the operating sequence step by step for students.
