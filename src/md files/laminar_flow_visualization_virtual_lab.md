# Laminar Flow Visualization Virtual Lab

## Category
Educational Simulation / Fluid Mechanics

## Short Summary
Unity-based virtual lab for visualizing laminar streamlines, flow patterns, sources/sinks, obstacles, and flow behavior through changing cross-sections.

## Portfolio Positioning
Present this as a fluid mechanics educational simulation.

The project focuses on making normally invisible flow behavior visible and interactive.

## Main Learning Goal
The user should understand how laminar streamlines behave around boundaries, obstacles, sources, sinks, and geometry changes.

## What I Built
- 3D/2D virtual flow visualization scene
- Streamline rendering
- Interactive flow controls
- Obstacle / shape interaction
- Source and sink style flow behavior
- Cross-section flow visualization
- Guided experiment UI
- Real-time visual feedback for flow pattern changes

## Simulated Components
- Transparent flow channel
- Streamline/dye injection points
- Flow source
- Flow sink/drain
- Obstacles or shape inserts
- Flow control valve
- Measuring grid
- Experiment UI panel

## Simulated Behaviors
- Smooth laminar streamline movement
- Streamlines bending around obstacles
- Flow contraction and expansion through cross-section changes
- Source/sink flow pattern visualization
- Flow-rate changes affecting streamline density/speed
- Visual comparison of different flow configurations

## Feature Cards

### Streamline Visualization
Shows clean, continuous laminar streamlines to make flow direction and behavior visible.

### Source / Sink Behavior
Visualizes ideal flow behavior around sources and sinks.

### Obstacle Interaction
Allows users to observe how streamlines bend around inserted shapes.

### Cross-Section Changes
Shows how flow behaves when channel geometry changes.

### Interactive Flow Control
Lets users change flow rate and observe immediate visual feedback.

### Guided Fluid Experiment
Presents the experiment as a student-friendly virtual lab workflow.
