# Bernoulli Principle Virtual Lab

## Category
Educational Simulation / Fluid Mechanics

## Short Summary
Unity-based virtual lab for visualizing Bernoulli’s principle through venturi flow, pressure distribution, velocity changes, and real-time measurement feedback.

## Portfolio Positioning
Present this as an engineering education simulation, not a game.

This project helps students understand the relationship between:
- flow velocity
- static pressure
- total pressure
- pipe / venturi cross-section
- flow rate

## Main Learning Goal
The user should understand that as the flow area changes, velocity and pressure change accordingly.

The simulation should visually show how fluid speeds up in narrow sections and how pressure readings change along the venturi.

## What I Built
- 3D virtual lab scene for Bernoulli experiment
- Venturi tube / flow channel visualization
- Fluid movement visualization
- Pressure tapping / manometer-style readings
- Interactive flow-rate control
- Real-time calculated values
- Graph or table for pressure/velocity comparison
- Guided experiment steps for students

## Simulated Components
- Venturi tube / nozzle
- Water flow path
- Inlet and outlet sections
- Pressure tapping points
- Manometer tubes or pressure indicators
- Flow control valve
- Measuring scale / reading points
- Hydraulic bench or water supply representation
- Experiment UI panel

## Simulated Behaviors
- Flow moving through different cross-section areas
- Velocity increasing in narrow sections
- Static pressure decreasing at high-velocity regions
- Pressure readings changing across tapping points
- Flow-rate changes affecting all measurements
- Comparison between theoretical and measured values

## Feature Cards

### Venturi Flow Visualization
Shows water movement through a changing cross-section to make velocity changes visible.

### Pressure Distribution
Displays static pressure readings along multiple points of the venturi.

### Flow Rate Control
Allows users to change inlet flow rate and observe the resulting pressure/velocity changes.

### Real-Time Calculations
Shows calculated values for velocity, pressure head, and total head.

### Guided Experiment
Walks students through the Bernoulli verification process step by step.

### Data Table / Graph
Displays readings across each section for comparison and lab reporting.
