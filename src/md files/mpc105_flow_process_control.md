## Module Overview Copy
MPC105 Flow Process Control is a Unity digital-shadow module for studying PID-based liquid-flow regulation. The simulation visualizes a pump-driven flow loop with turbine flowmeter feedback, rotameter reading, and proportional valve disturbance.

The module allows users to tune controller gains and observe how flow-rate response changes in real time. It is designed for educational experiments where students compare open-loop, closed-loop, P, PI, and PID behavior.

## Feature Cards

### Flow Loop Visualization
Shows water movement through the tank, pump, and transparent pipe system.

### Flowmeter Feedback
Represents turbine flowmeter measurement as closed-loop feedback.

### Rotameter Reading
Displays visual flow-rate indication for comparison and validation.

### Valve Disturbance
Simulates proportional valve disturbance by changing flow passage.

### PID Tuning
Allows users to compare P, PI, and PID flow-control behavior.

### Response Analysis
Displays flow response, error, control signal, and performance metrics.
